Curso de Programación Web

Sobre el proyecto :

Actualmente cursando la asignatura de Programación Web, en la cual desarrolla competencias fundamentales para la creación de páginas web modernas. Este curso le ha permitido comprender la estructura y el funcionamiento de los sitios web, así como aplicar buenas prácticas en el desarrollo frontend.

Durante el desarrollo del curso, ha trabajado en la creación de páginas web utilizando tecnologías como HTML5 y CSS3, aprendiendo a estructurar contenidos, diseñar interfaces y organizar información de manera clara y accesible.



Logros obtenidos :

  • HTML5: estructura básica de una página web
  • Etiquetas semánticas (header, nav, section, article, footer)
  • Enlaces relativos y absolutos
  • Imágenes y contenido multimedia
  • Formularios básicos
  • CSS3: estilos, colores, fuentes y diseño
  • Uso de Flexbox para diseño de páginas
